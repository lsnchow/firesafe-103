# How to Run and Setup Virtual Environent to Run Backend

## Running the Script

### On Windows
Run the `s.bat` script:
```cmd
s.bat
```

### On macOS/Linux
Run the `s.sh` script:
```bash
./s.sh
```

## Setting Up the Project

### Make new Terminal Tab

1. Install dependencies:
    ```bash
    npm i
    ```

2. Start the development server:
    ```bash
    npm run dev
    ```
